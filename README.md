# 🌌 Yu-Gi-Oh! Multiverse

O **Multiverso de Yu-Gi-Oh!** é o hub definitivo para fãs da franquia.  
Explore cronologias, histórias e todos os personagens em um único lugar.

Mais do que uma enciclopédia, o projeto permite **crossovers inéditos**, criando interações entre duelistas de diferentes gerações e simulando duelos épicos.

> Todo o universo de Yu-Gi-Oh! reunido em uma única aplicação.

---

## 🚀 Funcionalidades

### 📚 Enciclopédia Completa
Um catálogo abrangente com personagens das seguintes sagas:
- Duel Monsters  
- GX  
- 5D's  
- ZEXAL  
- ARC-V  
- VRAINS  
- SEVENS  
- GO RUSH!!  
- The Dark Side of Dimensions (DSOD)

Inclui:
- Cartas áses  
- Descrições detalhadas  
- Dados dos personagens  

---

### 🕰️ Cronologia e Lore
- Linha do tempo completa do universo  
- Eventos principais de cada geração  
- Organização narrativa entre séries  

---

### 🌐 Crossovers Interdimensionais
- Interações entre personagens de diferentes épocas  
- Sistema dinâmico de diálogo  
- Possibilidade de encontros nunca vistos no anime  

---

### ⚔️ Simulador de Duelos
Simule batalhas com diferentes mecânicas:
- Fusão  
- Sincro  
- Xyz  
- Pêndulo  
- Link  
- Maximum / Rush Duels  

---

## 🛠️ Tecnologias Utilizadas

- **HTML5 / CSS3**  
  Estrutura e estilização da interface

- **JavaScript (Vanilla)**  
  Lógica principal, simulações e interações

- **JSON**  
  Banco de dados local contendo:
  - IDs  
  - Sprites  
  - Informações dos personagens  
  - Falas  

---
